# Day 1 – Hour 3 (12:00 AM – 1:00 AM)

## Objective
To build a working prototype of the VC02 voice-controlled home automation system.

## Work Done
- Built hardware prototype on breadboard
- Connected VC02 module with microphone and speaker
- Connected 3 LEDs using relay (or direct testing setup)
- Completed firmware build and uploaded to VC02
- Implemented voice commands to control LEDs

## Circuit Implementation
- VC02 GPIO pins connected to control LEDs
- Power supplied using 5V module
- Microphone used for voice input
- Speaker used for audio response

## Result
- Successfully controlled 3 LEDs using voice commands
- System responded correctly to ON/OFF commands
