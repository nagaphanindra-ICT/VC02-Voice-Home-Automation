# Day 1 – Hour 2 (11:00 PM – 12:00 AM)

## Objective
To understand VC02 configuration, GPIO control, and voice command mapping.

## Work Done
- Studied VC02 configuration platform (voice SDK interface)
- Understood how voice commands are mapped to GPIO outputs
- Observed control logic for lights and fan
- Learned how HIGH/LOW signals control appliances

## Technical Understanding
- VC02 allows custom voice commands
- Each command is linked to GPIO pins
- Example:
  - Light ON → GPIO HIGH
  - Light OFF → GPIO LOW
- Multiple appliances can be controlled using different GPIO pins

## Learning
- VC02 does not require microcontroller (direct control)
- System works using predefined voice commands
- GPIO signals directly control relay module
